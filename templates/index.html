NAME
    Elimage: CLI Image Pastebin

SYNOPSIS
    curl -F 'name=@/path/to/image' {{ url }}

DESCRIPTION
    Paste image files with command line. You can give -F more times to paste
    more images.

EXAMPLES
    $ curl -F 'name=@screenshot.png' {{ url }}
    http://elimage.edisonnotes.com/97/2d8490849c8fab885218b478cc715ab0f3f4a9
